PyMOL

超便利な立体構造可視化ツール

PyMOL は、AlphaFold などを使って 作成したタンパク質立体構造を可視化できるソフトウェア。 様々な角度から立体構造を見ることが可能であり、美しい グラフィックも特徴である。

オープンソース版とバイナリ版が用意されており、バイナリ版は HP から 簡単にインストールできるが、アプリケーションを開くたびにライセンス認証を求められるわずらわしさがある。
オープンソース版は anaconda や homebrew を使ってインストールすることができ、ライセンス認証が 永続的に不要である。

mac の場合： brew install pymol


一方で、オープンソース版では後述の PyMod プラグインがインストールできない（少なくとも私が mac で試したときはそうだった）など、 不便な面もある模様。個人的には、普段使いとしてはオープンソース版を使用し、プラグインを使用するときはバイナリ版を 使用するのがベスト。

よく使うコマンド

PyMOL を開くと、External GUI と Viewer の2画面が表示される。
Viewer 画面には読み込んだ立体構造が表示され、さらに右側のオブジェクトパネルで表示形式や表示項目の 調整が可能である。
External GUI 画面では、コマンドラインによる操作ができる。
マウス操作とコマンドライン操作を組み合わせることで、より効果的な可視化やデータの出力が可能である。

よく使う操作：

・オブジェクトパネル

  • Show → cartoon/sticks etc.
    • （タンパク質の描画方法を指定。 cartoon はリボン状の描画で、二次構造が分かりやすい。sticks はアミノ酸の種類や方向が視覚的によくわかる。その他、複数の描画方法 show から指定できる。）
  • Display (or 右下) → Sequence
    • 立体構造を表示しているタンパク質のアミノ酸配列を表示する。任意のアミノ酸や配列の一部を選択すると、立体構造上のどの部分に該当するか 表示される。指定した部分のみ色を変えたり、描画方法を変えたりすることも可能。
  • Action → align → structure name
    • 表示している構造同士を重ね合わせる。重ね合わせの対象として同じタンパク質を指定すれば、複数タンパク質の重ね合わせも可能。 （ただし RMSD value や aligned sequences は1対1のみ表示。）
  • Action → preset → ligands (ligand sites)
    • （リガンドが立体構造上にある場合）リガンドと相互作用するアミノ酸サイトが表示される。

    ・コマンドライン
  • align structure_name_1, structure_name_2, (object= a)
    • 2つの構造を重ね合わせる操作。ただし1対1しかできない。
  • super structure_name_1, structure_name_2, (object= a)
    • 2つの構造を、立体構造をもとに重ね合わせる操作。ただし1対1しかできない。
  • label location（アミノ酸をクリックすればわかる）, "AA name"
    • アミノ酸名を表示して画像を出力したいときなどに使える。
  • set label_size, 20
    • ラベルのサイズを調整できる。他にもフォントの調節なども可。

    その他、様々な調整が可能。PyMOLWiki に 世界中の人が情報を蓄積しているので、大抵のやりたい操作は調べれば出てくる。

    絶対使うべき！ PyMOL プラグイン

    ・PyMod: タンパク質の Sequence Alignment、Multiple Structural Alignment、立体構造予測、構造間距離の算出など、 タンパク質構造解析関係の一通りのことができちゃうツール。マニュアル が非常に充実しているので参照されたし。

    ・CAVER: チャネルを予測するツール。PyMOL プラグインとして用意されており、PyMOL 上で可視化しながら 予測が可能なので便利…なのだが、Mac Desktop ではプラグインが使えず（チップの問題か？）
    → やむなく CAVER Analyst を利用
