MVD

タンパク質-リガンド相互作用を予測する "MVD"

Molegro Virtual Docker (MVD) は タンパク質-リガンド相互作用を予測することができる統合プラットホームである。
分子のプレパレーションからタンパク質のポテンシャルバインディングサイトの決定、リガンドのバインディングモードの予測まで、 ドッキングプロセスの全てを取り扱うことができる（HPより)。
一年契約の有償ライセンス（アカデミックライセンス）があり、購入すると MVD, MDM （MVDで予測した結果の解析に使用するソフトウェア）、英語版&日本語版のマニュアルを入手できる。
日本語のマニュアルがあるのでかなり使いやすいツールとなっている（ところどころ直訳でわかりにくいところがあるので、適宜英語版も参照することをお勧めする）
30日間の無料お試しもできる。

* マニュアル見てみたい場合はお知らせください（無料お試し版を申し込む場合はマニュアルもダウンロードできるはず）

基本的な使い方（ドッキングシミュレーション）

基本的にはマニュアルの2章に書いてあるチュートリアルに従って行えばドッキングシミュレーションできる。
ここでは簡単に説明する。

1. リガンドとタンパク質のファイルを準備（ファイル形式は PDB, MOL2, SDF など）
2. File -> Import Molecules #MVD でファイルを開く
3. Preparation -> Assign All Below -> Always #全てのpreparationをMVDに依存
4. Preparation -> Detect Cavities #タンパク質内のバインディングサイトを予測
5. View -> Reset View -> View -> Docking View #ドッキングシミュレーション用の画面表示に切り替え
6. Docking -> Docking Wizard #ドッキングのための設定開始
7. Binding Site -> Origin -> Cavity... #ドッキング予測をしたいバインディングサイトを選択 次のタブ2つくらいは変更の希望がなければそのまま進む
8. Errors and Warnings #潜在的な問題があれば警告が出る
9. Output directory #出力結果を保存するディレクトリと保存ファイル名を指定
10. Start -> Close #シミュレーション開始　指定したディレクトリ内にファイルが作成される

基本的な使い方（シミュレーション結果の解析）

シミュレーション結果を解析する（MVD、MDM、PyMOLを使用）

1. File -> Import Docking Results (*.mvdresults) ...　　　#ドッキング結果をインポート
2. Pose Organizer #予測されたリガンドのポーズ（デフォルト5ポーズ）のうち、表示したいものを選択
3. Pose Organizer -> Settings #ドッキングスコアなどの結果を表示
4. Ligand Map #リガンドとタンパク質間の相互作用を表示

その他、指定したアミノ酸サイトの置換、アフィニティ計算（MDMも必要）なども計算可能
